### Core Concept: What is a Crypto-Dividend-Tracker?

First, let's define the "dividend" in the crypto context. It's not a traditional corporate dividend. Instead, it refers to:

1.  **Staking Rewards:** Earnings from Proof-of-Stake (PoS) blockchains (e.g., ETH, ADA, DOT).
2.  **DeFi Yields:** Rewards from providing liquidity in Decentralized Exchanges (DEXs) like Uniswap or lending on platforms like Aave.
3.  **Airdrops & Hard Forks:** Distributions of new tokens to existing holders.

A **crypto-dividend-tracker** is a software platform (often a dApp or a centralized service) that automatically aggregates, tracks, and manages these various forms of yield-generating activities across multiple wallets and exchanges.
